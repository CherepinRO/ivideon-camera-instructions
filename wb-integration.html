Инструкция по интеграции

Пошаговое руководство по настройке камер Ivideon для работы с пунктами выдачи заказов Wildberries.

Шаг 1. Создание группы камер ПВЗ в Ivideon

1.1

Кликнуть на 3 вертикальные точки, далее "Добавить группу"

1.2

Назвать группу согласно шестизначному ID конкретного пункта выдачи (только цифры, не допускаются другие символы). Кликнуть «Добавить группу»

1.3

Выбрать камеры, принадлежащие данному ПВЗ, и кликнуть «Сохранить». После того как камеры перенеслись в правое поле, закрыть вкладку

Шаг 2. Верно назвать камеры

По регламенту, в ПВЗ Wildberries должно быть 3 камеры:

  • Зона выдачи – обзор действий администратора и выдачи товара
  • Обзорная – обзор происходящего в пункте, фиксирует отгрузку и прием товаров
  • Склад – обзор действий на складе
2.1

Задать название камеры согласно ее расположению. Назвать в таком формате все действующие камеры

2.2

Нажать 3 точки в правом нижнем углу камер, затем выбрать "Переименовать"

Шаг 3. Передать доступ на мастер-аккаунт Wildberries

3.1

Кликнуть по 3-м вертикальным точкам в поле «Мои камеры». Кликнуть «Передать права»

3.2

Выбрать нужную группу. Нажать «Продолжить»

3.3

Выбрать из списка передачи прав только «Живое видео», «Архив» и «События и уведомления». Кликнуть «Продолжить»

3.4

Внести в поле адрес мастер-аккаунта Wildberries:

video_wb.point@wildberries.ru

Кликнуть «Передать права»
